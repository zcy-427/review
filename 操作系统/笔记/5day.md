# 虚拟内存

虚拟内存是操作系统利用外存作为内存的扩充，使用户感觉可用内存空间比实际物理内存更大的技术。
简单说：

```text
内存不够，磁盘来凑。
```

其实现依赖两个基础：

```text
局部性原理
离散分配
```

### 局部性原理

程序运行时，在一段时间内只会访问其中一部分代码和数据。

分为：

|类型|含义|
|---|---|
|时间局部性|刚访问过的内容可能很快再次访问|
|空间局部性|访问某个地址后，附近地址可能很快被访问|

例子：

- 循环语句体现时间局部性
- 数组顺序访问体现空间局部性

### 请求分页

虚拟内存常用的实现方式

```text
请求分页存储管理
```

意思：

> 程序运行时不必一次全部装入内存，只在访问某页时，如果该页不在内存，再把它调入。

### 缺页中断

如果访问的页面不在内存，就会发生：

```text
缺页中断
```

操作系统会：

```text
从外存把该页调入内存
```

如果内存满了，还要：

```text
选择一个页面淘汰
```

这就引出页面置换算法。
