# 内存管理


### 1.内存管理的主要功能

|功能|含义|
|---|---|
|内存分配|给进程分配内存空间|
|内存回收|进程结束后收回内存|
|地址转换|把逻辑地址转换成物理地址|
|内存保护|防止进程越界访问|
|内存扩充|利用虚拟存储技术扩大逻辑内存|

## 2.几个基础概念

###  逻辑地址

> 程序中使用的地址，也叫相对地址或虚地址。

简单说：

```text
程序自己看到的地址。
```

## 2. 物理地址

> 内存中实际的地址，也叫绝对地址。

简单说：

```text
真实内存里的地址。
```

## 3. 地址重定位

> 把逻辑地址转换为物理地址的过程。

公式大致理解：

```text
物理地址 = 起始地址 + 逻辑地址
```
