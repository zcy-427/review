# 数据库系统的基本概念

1.**数据库DB**
长期存储在计算机内、有组织、可共享的大量数据集合

2.**数据库管理系统DBMS**
位于用户和操作系统之间，用于定义、操纵、管理和控制数据库的软件系统

3.**数据库系统DBS**
由数据库、数据库管理系统、应用系统、数据库管理员和用户等组成的系统

- **DB = 数据本身**
- **DBMS = 管理数据库的软件**
- **DBS = DB + DBMS + 应用程序 + 用户 + DBA**

# 数据模型
**数据模型**是对现实世界数据特征的抽象，用来描述数据结构、数据操作和数据约束。

数据模型尝包含3部分：

|组成|含义|
|---|---|
|数据结构|数据怎样组织|
|数据操作|可以对数据做什么操作|
|完整性约束|数据必须满足什么规则|
